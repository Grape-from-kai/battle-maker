## 概要
格闘ゲーム「Street Fighter 5」のオフ対戦会アプリ

## 解決したい課題
背景として、今作はゲームセンターに筐体を置かなくなってしまったので、対戦する機会の減少
が起きていると想定。
格闘ゲームの対戦機会増加と、それに伴うコミュニティの形成の一助になることを目的にしています。

## 想定している利用者
今作をプレイしているゲーマー
